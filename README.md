# Endure Cooling System

This repo contains the firmware for the cooling system, along with instructions on how to build the firmware and communicate with it.

Take a look at the [simple uart config protocol](docs/SIMPLE_UART_CONFIG_PROTOCOL.md) specification to see how to configure the device using UART with a serialport.

For the Ethernet Driver a slightly modified version of the one found in https://github.com/Georges760/STM32_ENC28J60 under [Drivers/Ethernet](https://github.com/Georges760/STM32_ENC28J60/tree/master/Drivers/Ethernet)
